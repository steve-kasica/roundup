# Roundup documentation

Welcome to Roundup - a click-based data integration system for journalists. This documentation provides everything you need to know to get started, explore example workflows, and dive deeper into the system's features.

## Explore example workflows

Our collection of example workflows reproduces 17 real-world computational and data journalism workflows written by journalists at leading news organizations, including the New York Times, ProPublica, BuzzFeed News, and more. Each workflow was originally done in programming languages, such as Python or R, and these workflow tutorials show how to reproduce the same consolidated data table using Roundup's click-based interface.

### Stack-only workflows

- **[2018 Voter Registration](/workflows/2018-voter-registration/)**: This workflow analyzes Maryland voter registration data across multiple elections to identify registration trends, patterns of voter movement between counties, and demographic shifts in voter registration over time.

### Pack-only workflows

- **[2015 U.S. Refugee Resettlement](/workflows/2015-11-refugees-in-the-united-states/)**: This workflow analyzes U.S. refugee resettlement data by state, combined with a table of refugee arrivals by destination with a table of U.S. Census state population estimates to calculate per-capita refugee resettlement rates by state.

### Hybrid workflows

- **[2018 Crime and Heat Analysis](/workflows/2018-05-31-crime-and-heat-analysis/)**: This workflow analyzes the relationship between crime and heat in St. Louis, Missouri, by joining a table of violent crime incidents with a table of daily maximum temperatures.
