# Roundup documentation

Welcome to Roundup - a click-based data integration system for journalists. This documentation provides everything you need to know to get started, explore example workflows, and diving deeper into the system's features.

## Quick Links

- **[Quick Start Guide](guides/quick-start/)** - New to Roundup? Start here for a beginner-friendly introduction to using the application.
- **[Explore example workflows](guides/example-workflows/)** - Browse our collection of **reproducible computational and data journalism workflows** published by leading news organizations.
- **[GitHub Repository](https://github.com/steve-kasica/open-roundup)** - Source code
- **[Issue Tracker](https://github.com/steve-kasica/open-roundup/issues)** - Report bugs or request features
- **[Discussions](https://github.com/steve-kasica/open-roundup/discussions)** - Ask questions or discuss ideas
